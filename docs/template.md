## 📦 インストール方法

### GitHub Release から

1. このリリースから `hakoview.vsix` をダウンロード
2. VS Code を開く
3. 拡張機能を開く（Ctrl+Shift+X / Cmd+Shift+X）
4. 「...」メニュー → 「VSIXからのインストール...」をクリック
5. ダウンロードした `hakoview.vsix` ファイルを選択

### VS Code Marketplace から

- 拡張機能で「Hako View」を検索
- 「インストール」をクリック

### OpenVSX Registry から

- https://open-vsx.org/extension/gen4438/hakoview にアクセス
- 「インストール」をクリック

## GitHub Release 作成手順

1. https://github.com/gen4438/HakoView/releases にアクセス
2. 「Draft a new release」をクリック
3. タグバージョン: `vX.Y.Z`（例: `v0.1.0`）
4. リリース タイトル: `Release vX.Y.Z` または `Hako View vX.Y.Z`
5. 説明: 上記のテンプレートを参照して記入
6. **ファイルを添付**:
   - `dist/hakoview.vsix` をドラッグ&ドロップ
7. 「最新リリースとしてマーク」をチェック
8. 「Save as draft」（下書き保存）または「Publish release」（公開）をクリック

## 公開手順

1. Draft として保存した場合:
   - 「Edit」をクリックして編集
   - 「Publish release」をクリック

2. 「Publish release」をクリック:
   - ワークフロー `publish.yml` が自動的にトリガー
   - マーケットプレイス設定があれば自動公開

## ベストプラクティス

- **絵文字を活用**: 読みやすくするため絵文字を使用
- **変更内容を明確に**: ユーザーが何が変わったかを理解できるように
- **既知の問題を明記**: ユーザーが問題を報告することを防ぐ
- **インストール手順を記載**: 新規ユーザーをサポート
- **貢献者に感謝**: コミュニティを大事にする
